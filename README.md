# CMDUtility

Windows PowerShell/CMD утилита - индикаторы загрузки для командной строки. Запустите утилиту через командную строку с одной из команд.

Commands:

        hepl - display hepl
        
        circle - run circle ainmation
        
        circle - run dots ainmation
        
        arrow - run arrow ainmation

Flags:

        -d - set duration in seconds
